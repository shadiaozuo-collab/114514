# Zsd 网游论坛 —— SillyTavern 沉浸式论坛脚本

> **把单调的聊天窗口变成多板块沉浸式论坛！** 现实讨论、游戏攻略、赛事战报、报纸期刊……AI 自动根据剧情生成帖子和评论，让你的世界真正"活"起来。

---

## ✨ 功能一览

| 功能 | 说明 |
|------|------|
| 📝 **多板块论坛** | 支持论坛 / 赛事 / 报纸三种板块类型，最多5个板块，独立帖子流 |
| 🤖 **AI 自动生成** | 每轮对话后自动根据剧情生成帖子，支持倒计时提醒和跳过原因反馈 |
| 🎨 **8 种主题** | 经典暗黑 / 赛博朋克 / 蒸汽波 / 终端绿 / 简约白 / 羊皮纸 / 暗金 / 自定义配色 |
| 💬 **智能注入** | 自动将最新论坛内容注入酒馆对话上下文，让AI"看到"论坛动态 |
| ❤️ **点赞热度系统** | AI 生成模拟点赞数（0-999），用户可手动点赞/取消，支持关闭 |
| 🏆 **赛事战报** | 赛事板块自动生成比赛战报，含对阵、比分、胜者、MVP |
| 📰 **报纸期刊** | 报纸板块自动生成完整一期报纸，含头条、栏目、主编寄语 |

| 🛡️ **截断保护** | AI回复过短/空回时自动跳过生成，避免无效内容污染论坛 |

## 📦 安装方法

### 方式一：直接导入（推荐）

在 SillyTavern 的**前端界面/脚本**中，添加以下代码：

```javascript
import 'http://你的服务器地址/Zsd网游论坛/index.js'
```

> 服务器需要指向本项目的 `dist/` 目录。可用 `python -m http.server 5500` 或任意静态文件服务器（需开启CORS）。

### 方式二：本地导入

把打包后的 `dist/Zsd网游论坛/index.js` 放到 SillyTavern 可访问的路径，直接导入即可。

导入成功后，酒馆界面会出现一个 **「论坛」** 按钮，点击即可打开悬浮窗。也可在扩展菜单（左下角菜单）中找到「网游论坛」入口。

## 🖱️ 界面操作

### 打开论坛

点击酒馆按钮栏的 **「论坛」** 按钮，或扩展菜单中的 **「网游论坛」**，屏幕弹出论坛悬浮窗。

- **拖拽移动**：按住顶部标题栏拖动（桌面端）
- **关闭**：点击右上角 × 或再次点击「论坛」按钮
- **注意**：关闭面板只是隐藏界面，**后台自动生成仍在运行**

### 浏览帖子

```
┌─────────────────────────────────────────────┐
│  Z论坛        [现实A] [赛事] [报纸] [游戏B]   │  ← 顶部标签切换板块
├─────────────────────────────────────────────┤
│  AI 正在生成内容，请稍候… (12s) [终止]        │  ← 生成中提示条
├─────────────────────────────────────────────┤
│  【组队】今晚有没有刷战场的？                  │  ← 帖子列表
│  固定队少个法师或者弓手...                    │
│  💬 3  ❤️ 128  剑影无痕 · 第三天·午后         │
├─────────────────────────────────────────────┤
│  关于新出的魔法石合成系统...                   │
│  昨晚砸了五万金币进去...                      │
├─────────────────────────────────────────────┤
│  [🔄刷新] [⚙设置] [AI生成] [✏发帖]           │  ← 底部操作栏
└─────────────────────────────────────────────┘
```

- 点击帖子标题 → 进入帖子详情页，查看完整内容、评论、点赞
- 点击顶部标签 → 切换板块
- 点击 **「❤️」** → 点赞/取消点赞帖子
- 点击 **「🔄 刷新」** → 从变量重新加载数据并清理孤儿帖子

### 发帖

点击底部 **「✏ 发帖」** 按钮，填写标题、内容、作者ID和时间戳，即可手动发布。手动帖子不会被自动清理删除。

### AI 生成帖子

点击底部 **「AI生成」** 按钮：
- 选择要生成的板块（单个 / 全部）
- 选择生成模式：独立生成（逐个调用，更稳定）或合并生成（单次调用，更快）
- 可选输入话题方向（留空则由AI自由发挥）
- 点击生成，等待AI完成

生成过程中顶部会显示进度条和计时器，可随时点击「终止」取消。

## ⚙️ 设置详解

点击论坛右上角的 **⚙ 齿轮图标** 打开设置面板。

### 基本设置

| 设置项 | 说明 |
|--------|------|
| **论坛名称** | 标题栏显示的名称 |
| **板块名称/类型** | 每个板块的显示名称和类型（论坛/赛事/报纸），最多5个板块 |
| **玩家论坛ID** | 当前主角在论坛中的ID，AI生成涉及主角时会使用此ID |
| **去中心化模式** | 开启后，大部分帖子由"路人"发起，只有少数围绕主角 |
| **显示点赞数（热度）** | 开关点赞系统 |
| **自动AI回复** | 用户手动发帖/评论后，自动生成AI评论回复 |

### 背景设置

| 设置项 | 说明 |
|--------|------|
| **背景图片** | 点击上传本地图片，支持 JPG/PNG/GIF |
| **背景不透明度** | 0-100%，控制背景遮罩浓淡 |
| **背景模糊度** | 0-20px，开启毛玻璃效果 |

### 提示词设置

| 设置项 | 说明 |
|--------|------|
| **板块系统提示词** | 每个板块独立的AI生成风格控制。点击「风格预设」下拉可一键切换14种世界观 |

### AI 生成参数

| 设置项 | 说明 |
|--------|------|
| **帖子数量提示** | 每次生成大约多少个帖子（3-5个为宜） |
| **评论数量提示** | 每个帖子带多少条评论 |
| **AI回复最短字数** | AI回复低于此字数时跳过生成（0=不检测） |

### 上下文注入

| 设置项 | 说明 |
|--------|------|
| **启用注入到聊天** | 开启后，论坛最新内容会自动注入酒馆对话 |
| **注入帖子数** | 每板块最多注入几条帖子到对话 |
| **注入聊天记录数** | 发送给AI时保留多少轮对话历史 |

### 自动生成

| 设置项 | 说明 |
|--------|------|
| **自动生成帖子间隔** | `0`=关闭，`-1`=每轮都生成，`3`=每3轮AI回复生成一次 |
| **AI回复最短字数** | AI回复低于此字数时跳过生成（防止截断/空回） |
| **自动生成板块** | `全部`=所有板块一起生成，或指定单个板块 |
| **多板块自动生成模式** | `独立生成`（逐个调用，更稳定）/ `合并生成`（单次调用，更快） |

**自动生成的工作流程：**

```
你发送消息 → AI开始生成 → AI生成完成
                                      ↓
                              [GENERATION_ENDED 触发]
                                      ↓
                    ┌─────────────────┴─────────────────┐
                    ↓                                   ↓
            [门控检查]                           [字数检测]
            · 不是quiet生成？                     · AI回复 > 阈值？
            · 不是dryRun？                        ↓
            · 不是后台触发？              [计数达到间隔？]
                    ↓                                   ↓
                    └─────────────────┬─────────────────┘
                                      ↓
                              [调用AI生成帖子]
                              同时写入当前AI消息extra
                                      ↓
                              [帖子添加到论坛]
                                      ↓
                              [重新注入上下文]
```

> **关闭面板不影响自动生成！** 隐藏后后台仍然监听 `GENERATION_ENDED` 事件，达到间隔时自动触发。

### 自动清理

| 设置项 | 说明 |
|--------|------|
| **启用自动清理** | 开启后，帖子数超阈值时自动删除最旧的帖子 |
| **清理阈值** | 每个板块最多保留多少条帖子 |

> 手动添加的帖子（`sourceMessageIndex === undefined`）不受清理影响。只有AI生成的帖子会被自动清理。

### API 配置

支持完全独立于酒馆主设置的外部 API：

| 设置项 | 说明 |
|--------|------|
| **模型名** | 如 `gpt-4o`、`claude-3-5-sonnet-20241022`。支持下拉选择或手动输入 |
| **代理预设** | 使用酒馆已保存的代理配置（优先） |
| **API 地址** | 自定义 API 端点 |
| **API Key** | 自定义 API Key |
| **API 源** | 如 `openai`、`custom` |

**API 预设管理**：
- 填好配置后输入预设名称，点击 **「保存预设」**
- 之后可在下拉框中选择预设，点击 **「应用预设」** 一键加载
- 不需要的预设可点击 **「删除」**

> API配置保存在**全局变量**中，切换对话后仍然保留。论坛数据保存在**对话变量**中，每个聊天独立。

## 🎯 使用建议

### 场景1：纯自动模式（懒人必备）

1. 设置「自动生成帖子间隔」为 `-1`（每轮都生成）或 `3`（每3轮）
2. 设置「自动生成板块」为「全部」
3. 为每个板块选择合适的「风格预设」
4. 关闭设置，正常聊天即可

论坛会后台自动生成帖子，AI在对话中也能感知论坛动态。

### 场景2：半自动模式（追求质量）

1. 设置「自动生成帖子间隔」为 `0`（关闭自动）
2. 遇到关键剧情节点时，手动点击「AI生成帖子」
3. 在弹窗中输入具体话题方向（如"公会战爆发"）
4. 生成后查看效果，不满意可以删除/Swipe后重新生成

### 场景3：赛事+报纸联动

1. 创建4个板块：现实讨论(论坛) / 游戏论坛(论坛) / 巅峰对决(赛事) / 艾泽拉斯日报(报纸)
2. 为赛事和报纸板块分别选择对应风格预设
3. 开启自动生成，设置间隔为 `2-3`
4. 剧情推进时，论坛、赛事、报纸会同步更新，形成完整的世界观生态

### 场景4：破甲失败时的处理

如果AI仍然拒绝生成：
1. 打开设置 → 提示词设置 → 破甲提示词
2. 根据你的模型调整破甲内容（不同模型对破甲敏感度不同）
3. 或尝试关闭「带入酒馆预设上下文」（如果之前手动开启了）
4. 或切换到更强/更开放的模型

## ↩️ 回退同步说明

当你 **删除** 或 **Swipe** 一条AI回复时，该条消息对应的论坛帖子会自动消失。

**为什么？**

每轮生成的帖子都会"绑定"到生成它的那条AI消息上（通过 `extra.zsdForumGeneration`）。消息不在了，帖子自然跟着回退——就像你真的在"撤销"那轮对话的副作用。

**手动添加的帖子不受影响。**

**刷新按钮的作用：**

如果因为数据异常导致面板显示错误（空白、错位、卡死），点击左上角的 **🔄 刷新** 按钮：
1. 从酒馆变量重新加载全部数据
2. 扫描所有AI消息，清理无引用的"孤儿帖子"
3. 恢复面板正常显示

## ❓ 常见问题

**Q1：为什么关闭了论坛面板，自动生成还在跑？**

这是**设计如此**。关闭面板只是隐藏界面，后台的自动生成监听仍然运行。这样你不需要一直开着论坛窗口也能自动更新。如果确实想完全停止，把「自动生成帖子间隔」设为 `0`。

**Q2：为什么设置了自动生成，但有时候没生成？**

检查 Console 日志或 toast 提示：
- 「AI回复过短，跳过自动生成」→ 触发了字数保护
- 「跳过本轮 GENERATION_ENDED (type=quiet)」→ 这是后台quiet生成（如MVU），被正确过滤
- 「倒计时：还剩 N 轮」→ 间隔还没到

**Q3：帖子可以永久保存吗？**

可以。所有帖子数据存储在酒馆**对话变量**中（以 `Z` 开头），随聊天记录一起保存。切换聊天文件时数据自动隔离。API配置存储在**全局变量**中，跨对话共享。

**Q4：删除对话楼层后帖子没回退？**

请确保你删除的是 **AI回复楼层**（不是用户楼层）。只有AI消息生成的帖子才会被追踪。如果还是没回退，点击刷新按钮手动触发清理。

**Q5：双板块/多板块同时生成质量会不会下降？**

对模型有一定要求。建议：
- 使用 GPT-4o、Claude 3.5+ 等较强模型
- 如果质量不稳定，可把「多板块自动生成模式」改为「独立生成」
- 或把「自动生成板块」改为单板块轮流生成

**Q6：论坛窗口挡住了其他界面怎么办？**

按住顶部标题栏拖拽到合适位置（桌面端）。手机端点击半透明遮罩即可关闭。

**Q7：Firefox 能用吗？**

论坛扩展目前**仅推荐 Chrome/Edge**。Firefox 在 SillyTavern 的 iframe 机制下存在兼容性问题，可能导致生成失败。

**Q8：为什么AI生成的帖子越来越重复？**

如果还遇到这个问题：
1. 检查是否关闭了「带入酒馆预设上下文」（默认已关）
2. 确保提示词中有 `<anti_repetition>` 模块（默认已带）
3. 尝试降低「注入帖子数」，减少历史上下文干扰
4. 换用更强的模型

## 🛠️ 技术信息

- **框架**：Vue 3 + Pinia + TypeScript + TailwindCSS
- **运行环境**：SillyTavern iframe（JS-Slash-Runner）
- **输出格式**：XML（`<post>` `<comment>` 标签）
- **打包命令**：`pnpm build`
- **开发命令**：`pnpm watch`（热重载）
- **产物路径**：`dist/Zsd网游论坛/index.js`
- **推荐浏览器**：Chrome / Edge（Firefox 存在兼容性问题）
