# Zsd 网游论坛 —— SillyTavern 沉浸式论坛脚本

> **把单调的聊天窗口变成双板块网游论坛！** 让你的角色在"现实讨论区"聊八卦，在"游戏论坛"刷攻略帖，自动生成、自动同步、零门槛安装。

---

## ✨ 功能一览

| 功能 | 说明 |
|------|------|
| 📝 **双板块论坛** | 板块A（现实向）+ 板块B（游戏向），独立帖子流，互不干扰 |
| 🤖 **AI 自动生成** | 每轮对话后自动根据剧情生成论坛帖子和评论，支持一次调用双板块同时产出 |
| 💬 **智能注入** | 自动将最新论坛内容注入酒馆对话上下文，让AI"看到"论坛动态 |
| 🎨 **6 种主题** | 经典暗黑 / 赛博朋克 / 蒸汽波 / 终端绿 / 简约白 / 自定义配色 |
| 🖼️ **本地上传背景** | 支持上传任意图片作为论坛背景，支持透明度+毛玻璃调节 |
| 🛡️ **截断保护** | AI回复过短/空回时自动跳过生成，避免无效内容污染论坛 |
| ↩️ **回退同步** | 删除或Swipe对话楼层时，对应生成的帖子自动跟着回退 |
| 🔧 **自定义 API** | 支持独立配置模型、代理、API地址、Key，也可保存多组预设快速切换 |
| 🧹 **自动清理** | 帖子数量超阈值时自动删除旧帖，防止数据膨胀 |

## 📦 安装方法

### 方式一：直接导入（推荐）

在 SillyTavern 的**前端界面/脚本**中，添加以下代码：

```javascript
import 'http://你的服务器地址/Zsd网游论坛/index.js'
```

> 服务器需要指向本项目的 `dist/` 目录。可以用 `python -m http.server 8080` 或任意静态文件服务器。

### 方式二：本地导入

把打包后的 `dist/Zsd网游论坛/index.js` 放到 SillyTavern 可访问的路径，直接导入即可。

导入成功后，酒馆界面会出现一个 **「论坛」** 按钮，点击即可打开悬浮窗。

## 🖱️ 界面操作

### 打开论坛

点击酒馆按钮栏的 **「论坛」** 按钮，屏幕右上角弹出论坛悬浮窗。

- **拖拽移动**：按住顶部标题栏拖动
- **关闭**：点击右上角 × 或再次点击「论坛」按钮

### 浏览帖子

```
┌─────────────────────────────┐
│  Z论坛        [现实A] [游戏B] │  ← 顶部标签切换板块
├─────────────────────────────┤
│  【组队】今晚有没有刷战场的？  │  ← 帖子列表
│  固定队少个法师或者弓手...    │
│  💬 3    剑影无痕 · 第三天·午后│
├─────────────────────────────┤
│  关于新出的魔法石合成系统...   │
│  昨晚砸了五万金币进去...      │
├─────────────────────────────┤
│  [⚙ 设置]   [AI生成帖子] [发帖]│  ← 底部操作栏
└─────────────────────────────┘
```

- 点击帖子标题 → 进入帖子详情页，查看完整内容和评论
- 点击 **「现实A」/「游戏B」** 标签 → 切换板块
- 点击 **「← 返回」** → 回到帖子列表

### 发帖

点击底部 **「发帖」** 按钮，填写标题、内容、作者ID和时间戳，即可手动发布一条帖子。

### AI 生成帖子

点击底部 **「AI生成帖子」** 按钮：
- 选择要生成的板块（A / B / 全部）
- 可选输入话题方向（留空则由AI自由发挥）
- 点击生成，等待几秒后帖子自动出现在列表顶部

## ⚙️ 设置详解

点击论坛右上角的 **⚙ 齿轮图标** 打开设置面板。

### 基本设置

| 设置项 | 说明 |
|--------|------|
| **论坛名称** | 标题栏显示的名称，如"Z论坛"、"艾泽拉斯酒馆" |
| **论坛A/B 名称** | 两个板块的显示名称，可自定义 |
| **玩家论坛ID** | 当前主角在论坛中的ID，AI生成涉及主角时会使用此ID |

### 主题风格

- **6 种预设主题**：一键切换整体配色
- **自定义主题**：选择「自定义」后，可分别调节背景色、卡片色、文字色、强调色

### 背景设置

| 设置项 | 说明 |
|--------|------|
| **背景图片** | 点击上传本地图片，支持 JPG/PNG/GIF |
| **背景不透明度** | 0-100%，控制背景遮罩浓淡 |
| **背景模糊度** | 0-20px，开启毛玻璃效果 |

### 提示词设置

| 设置项 | 说明 |
|--------|------|
| **A/B 板块提示词** | 控制该板块AI生成帖子的风格、主题方向 |
| **输出格式提示词** | 控制AI输出帖子的JSON格式和字段要求 |
| **用户ID生成规则** | 控制AI生成发帖人ID的风格 |
| **去中心化模式** | 开启后，大部分帖子由"路人"发起，只有少数围绕主角 |

### AI 生成参数

| 设置项 | 说明 |
|--------|------|
| **帖子数量提示** | 每次生成大约多少个帖子（3-5个为宜） |
| **评论数量提示** | 每个帖子带多少条评论 |
| **携带预设上下文** | 是否让AI生成时读取酒馆的角色卡/世界书/场景设定 |

### 上下文注入

| 设置项 | 说明 |
|--------|------|
| **启用注入到聊天** | 开启后，论坛最新内容会自动注入酒馆对话 |
| **注入帖子数** | 每板块最多注入几条帖子到对话 |
| **注入聊天记录数** | 发送给AI时保留多少轮对话历史 |

### 自动生成

| 设置项 | 说明 |
|--------|------|
| **自动生成帖子间隔** | `0`=关闭，`-1`=每轮都生成，`3`=每3轮AI回复生成一次 |
| **AI回复最短字数** | AI回复低于此字数时跳过生成（防止截断/空回） |
| **自动生成板块** | `全部`=双板块同时生成，`A`/`B`=只生成单板块 |

### 自动清理

| 设置项 | 说明 |
|--------|------|
| **启用自动清理** | 开启后，帖子数超阈值时自动删除最旧的帖子 |
| **清理阈值** | 每个板块最多保留多少条帖子 |

### API 配置

支持完全独立于酒馆主设置的外部 API：

| 设置项 | 说明 |
|--------|------|
| **模型名** | 如 `gpt-4o`、`claude-3-5-sonnet-20241022` |
| **代理预设** | 使用酒馆已保存的代理配置 |
| **API 地址** | 自定义 API 端点 |
| **API Key** | 自定义 API Key |
| **API 源** | 如 `openai`、`custom` |

**API 预设管理**：
- 填好配置后输入预设名称，点击 **「保存预设」**
- 之后可在下拉框中选择预设，点击 **「应用预设」** 一键加载
- 不需要的预设可点击 **「删除」**

> 如果不配置 API，论坛生成将使用酒馆当前的默认连接。

## 🧠 自动生成的完整流程

当你正常与AI对话时，论坛会在后台默默工作：

```
你发送消息 → AI开始生成 → AI生成完成
                                      ↓
                              [GENERATION_ENDED 触发]
                                      ↓
                    ┌─────────────────┴─────────────────┐
                    ↓                                   ↓
            [门控检查]                           [字数检测]
            · 不是quiet生成？                     · AI回复 > 阈值？
            · 不是dryRun？                        ↓
            · 不是后台触发？              [计数达到间隔？]
                    ↓                                   ↓
                    └─────────────────┬─────────────────┘
                                      ↓
                              [调用AI生成帖子]
                              同时写入当前AI消息extra
                                      ↓
                              [帖子添加到论坛]
                                      ↓
                              [重新注入上下文]
```

这意味着：**你只管正常聊天，论坛内容会随着剧情推进自动更新。**

## ↩️ 回退同步说明

当你 **删除** 或 **Swipe** 一条AI回复时，该条消息对应的论坛帖子会自动消失。

**为什么？**

因为每轮生成的帖子都会"绑定"到生成它的那条AI消息上。消息不在了，帖子自然跟着回退——就像你真的在"撤销"那轮对话的副作用。

**手动添加的帖子不受影响。**

## 🎯 使用建议

### 场景1：纯自动模式（懒人必备）

1. 设置「自动生成帖子间隔」为 `-1`（每轮都生成）
2. 设置「自动生成板块」为「全部」
3. 根据剧情阶段微调 A/B 板块的提示词
4. 关闭设置，正常聊天即可

论坛会每轮自动生成双板块帖子，AI在对话中也能感知论坛动态。

### 场景2：半自动模式（追求质量）

1. 设置「自动生成帖子间隔」为 `0`（关闭自动）
2. 遇到关键剧情节点时，手动点击「AI生成帖子」
3. 在弹窗中输入具体话题方向（如"公会战爆发"）
4. 生成后查看效果，不满意可以删除/Swipe后重新生成

### 场景3：自定义主题打造专属风格

1. 选择「自定义」主题
2. 配合背景图上传，调节透明度和模糊度
3. 微调四个颜色变量，打造专属视觉

## ❓ 常见问题

**Q1：为什么设置了自动生成，但有时候没生成？**

检查 Console 日志：
- 如果看到「AI回复过短，跳过自动生成」→ 说明触发了字数保护，AI这轮回复太短/被截断了
- 如果看到「跳过本轮 GENERATION_ENDED (type=quiet)」→ 这是后台quiet生成（如MVU变量解析），被正确过滤了

**Q2：帖子可以永久保存吗？**

可以。所有帖子数据存储在酒馆变量中（以 `Z` 开头），随聊天记录一起保存。切换聊天文件时数据自动隔离。

**Q3：删除对话楼层后帖子没回退？**

请确保你删除的是 **AI回复楼层**（不是用户楼层）。只有AI消息生成的帖子才会被追踪。如果还是没回退，检查 Console 是否有 `[网游论坛] 回退同步` 日志。

**Q4：手动发的帖子和AI生成的帖子混在一起怎么办？**

手动帖子没有字数限制，也不会被自动清理规则删除（除非开启自动清理且总数量超阈值）。建议定期手动整理，或提高清理阈值。

**Q5：双板块同时生成质量会不会下降？**

对模型有一定要求。建议：
- 使用 GPT-4o、Claude 3.5+ 等较强模型
- 如果质量不稳定，可把「自动生成板块」改为单板块轮流生成

**Q6：论坛窗口挡住了其他界面怎么办？**

按住顶部标题栏拖拽到合适位置。关闭后再打开会记住最后位置（在当前会话内）。

## 📝 变量清单

所有论坛相关变量以 `Z` 开头，可在酒馆变量面板中查看：

| 变量名 | 说明 |
|--------|------|
| `Zposts` | 帖子数据（A/B两个板块） |
| `ZforumName` | 论坛名称 |
| `Ztheme` | 当前主题 |
| `ZautoGenerateInterval` | 自动生成间隔 |
| `ZautoGenerateSection` | 自动生成板块 |
| `ZminReplyLength` | AI回复最短字数阈值 |
| `ZenableInjectToChat` | 是否注入上下文 |
| `ZbgImage` | 背景图Base64 |
| `ZcustomBg`/`ZcustomAccent`/`ZcustomText`/`ZcustomCardBg` | 自定义主题颜色 |
| `ZapiPresets` | API预设列表 |
